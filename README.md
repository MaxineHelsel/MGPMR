# MGPMR

### Instructions
Upon first boot MGPMR will create the necessary directory tree in its working directory. 
Close MGPMR and populate the ./Library/ folder with the card files of your choosing in the rarities of your choosing.
Reopen MGPMR and veryfy MGPMR has detected the cards in the ./Library/ directory.
Using WASD or Arrow keys, adjust the weights and parameters to your desired liking. (Use shift to fine tune weights)
Select "Generate Packs" to save parameters and generate the requested booster packs.
Generated packs will be saved in the ./Pack/(SessionTimestamp)/ directory.
